# Screenshots

Add your project screenshots here after running the project locally:

1. `dashboard.png` - Main dashboard with stats
2. `books.png` - Books list page
3. `issue.png` - Issue book page
4. `transactions.png` - Transactions page with fine details

## How to take screenshots:
1. Run `python manage.py runserver`
2. Open `http://127.0.0.1:8000`
3. Login and navigate to each page
4. Press `Windows + Shift + S` to take screenshot
5. Save in this folder with the names above
